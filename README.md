Developer Portfolio – Minh Mai

This is a modern, responsive personal portfolio built with Next.js, React, Tailwind CSS, and Framer Motion. It presents my background, skills, selected projects, and a functional contact form designed for professional outreach.

Live Site: https://itsminhmai.com

Overview

The portfolio showcases:

> A dynamic hero section with typing effects and resume access

> An About section detailing background, education, skills, and interests

> A Projects section featuring interactive project cards with animations, tech tags, and links

> A fully functional contact form powered by EmailJS

> Responsive design and smooth motion built with Framer Motion

> Social links and a simple footer

> The code follows a component-based structure, uses modern UI libraries, and prioritizes clarity, performance, and maintainability.

Tech Stack

> Frameworks: Next.js, React
> Styling: Tailwind CSS, shadcn/ui components
> Animation: Framer Motion
> Integrations: EmailJS, lucide-react, react-icons
> Other: Responsive design, custom motion variants, client-side email handling
